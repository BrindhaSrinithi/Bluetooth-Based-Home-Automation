# Bluetooth-Based-Home-Automation
This project demonstrates a simple Bluetooth-based home automation system that allows users to control household devices directly from their smartphones. It offers a convenient and user-friendly solution for operating appliances without needing to physically interact with switches.
## PROJECT DESCRIPTION:
The system uses an HC-05 Bluetooth module to establish a wireless connection between an Android smartphone and an Arduino Uno. Using a Bluetooth terminal or mobile application, the user can turn an LED (representing a light) and a DC motor (representing a fan) ON or OFF by sending simple commands from the phone. This project demonstrates the practical application of Bluetooth communication in making everyday tasks more convenient and accessible.
## COMPONENTS USED:
1. Arduino Uno
2. HC-05 Bluetooth Module
3. LED
4. DC Motor (Represents a Fan)
5. L293D Motor Driver IC Motor Driver Module
6. 220Ω Resistor (for LED)
7. Breadboard
8. Jumper Wires
9. USB Cable
10. Android Smartphone with a Bluetooth Terminal App (or a custom Bluetooth control app)

## FEATURES:
- Control lights using a smartphone.
- Turn a fan (DC motor) ON and OFF wirelessly.
- No physical contact with switches required.
- Simple and beginner-friendly wireless automation project.
- Real-time Bluetooth communication.

## APPLICATIONS:
- Home automation
- Smart lighting control
- Wireless fan control
- Bedroom and living room appliance control
- Convenience for elderly and physically challenged individuals
